# Flowdesk Dashboard (Frontend)

This is the **React frontend** for the **Flowdesk Task & Project Management System**. It connects to the backend API to manage projects and tasks with a user-friendly interface.

## Features

- View all projects and tasks  
- Add, edit, and delete projects  
- Add, edit, and delete tasks  
- Validation on forms for required fields  
- Confirmation modals for deletions  
- Navigate between projects and their tasks  

---

## Technology Stack

- **React** (v18) with TypeScript  
- **React-Bootstrap** for UI components  
- **Axios** for API calls  
- **React Router DOM** for navigation  
- **Bootstrap 5** for styling
